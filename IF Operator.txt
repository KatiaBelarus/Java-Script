         Условный Оператор IF 
  Условия состоят из двух частей - в первой части ты указываешь,что делать когда условие верное а во второй - когда неверное.
    ЕСЛИ..., ТО...

    ЕСЛИ - IF
   if (условие,действие)   
 >let a = 0;
  let liked = true;          let liked = true;
  if (liked)                 if (true)
      a = 5;                    a=5;
 <5
 >a;
 <5 

   Если изменить таким образом:
  
 >a=0;                       let liked = false;
  liked = false;             if(false)
  if(liked)                      a=5;
  a=5;
 >a;
 <0 

  Следующие значения всегда будут преобразованы в false - ноль, " " пустые кавычки, null, undefined, NaN,.

  Что делать,если нужно выполнить действие,когда значение FALSE - нужно использовать операторы сравнения.Пишем нужное сравнение,чтобы его результатом было да, тоесть true.

  Что делать,если нужно выполнить действие,если в условии FALSE - для этого нужно написать такое условие,чтобы оно возвращало true.


  Сокращения в условии могут быть двух типов:
  - сокращения при проверке на true
  - сокращения при проверке на false

     На TRUE - НЕОБХОДИМО ПРОПИСАТЬ ЗНАЧЕНИЯ В УСЛОВИИ,без оператора сравнения это значение будет пытаться переобразоваться к булевому.
    a=0;
    if(true)
    a=5;
  Если оно преобразуется в true, то условие выполнится.
  К true преобразуются все,кроме нуля, " ", null, undefined и NaN.

    На FALSE - суть сокращения в том,что == false
                                         === false    - мы можем заменить на !
    >a=0;
    <0                           liked = false;
    >liked=false;                if(liked=false)
    <false                            !liked
    >if(!liked)                          a=5;
         a=5
    <5
    >a;
    <5

    То же самое можно записать со ЗНАЧЕНИЯМИ,КОТОРЫЕ ВСЕГДА преобразуются в FALSE.

    !0 => 0==false => true
    !null => null==false => true
    !undefined => undefined==false => true
    !NaN => NaN==false => true
    !" => "==false => true

    По дефолту всегда выполняется только одно действие (=>...<=):
       a=0;
      => if(false) <=
           a=5;
           a=a + 10;        Тоесть,в конце будет 10


    Если нужно сделать 0 или более действий,то делаем блок с ними: для этого действия оборачиваем в { }
      >if(true) {              Если нужно ноль действий,то пишем пустые скобки:
        a=5;
        a=a + 10;                  >if(true) {
      }                             }
      <15
 
    Действия при неверном условии:

    if (условие) действие
    else - альтернативное действие

    Всегда выполняется ТОЛЬКО ОДНО условие: !! или if - или else  !!

    Если пишем с { },то действий может добавляться сколько угодно.


           УСЛОВИЯ - работают по принципу - какое условие первое подойдёт,такое и выполняется.А остальные пропускаются => ELSE IF
