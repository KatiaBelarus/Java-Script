              Конструкция SWITCH:
 Является альтернативой условному оператору IF.
    Структура => SWITCH (значение_для_сравнения) {
                 case 'с_чем_сравнивать_1';
                 действия 1;
                 break;
                 case 'с_чем_сравнивать_2':
                 действия 2;
                 break;
           }
 Значения для сравнения могут быть как конкретные,так и переменные.

      Значения для сравнения:                          С чем сравнивать:
 1) SWITCH (1)                                       1) case 1:
 2) SWITCH (true)                                    2) case true:
 3) SWITCH ('text')                                  3) case 'text':
 4) SWITCH (2.5)                                     4) case 2.5:
 5) SWITCH (undefined)                               5) case undefined:
 6) let variable = 5;                                6) let variable =2;
    switch(variable)                                    case variable ===2: 

 BREAK: используется для избежания перехода к следующему условию.

  В случае с IF используется ELSE
  В случае с SWITCH используется DEFAULT
     switch(значение_для_сравнения) {
        case 'с_чем_сравнивать_1':
        действия1;
        break;
        case 'с_чем_сравнивать_2':
        действия2;
        break;
        default:
           действия_когда_условия_не_подошли;
           break;
   }

       Конструкция без BREAK - тут мы попадаем в условие через "или" - выполнится или это условие (1), или то условие (2)
